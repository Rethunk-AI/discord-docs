# Content Policy & Moderation

## Reporting Policy Violations

Community documentation must comply with Discord community guidelines and our moderation standards. Report violations via:

**Primary:** Open a GitHub issue on this repository with the label `[VIOLATION]`.  
**Escalation:** Contact `@staff` in the Discord server for urgent moderation.

### What to Report
- Misinformation or outdated guidance in documentation
- Harmful advice or unsafe practices
- Offensive or exclusionary language
- Links or references to malicious content
- Copyright or attribution violations

### Response Timeline
- **Acknowledged:** Within 24 hours
- **Investigation:** Within 3 business days
- **Resolution:** Documented and published

## Content Standards

### Accuracy
- Information must be current and technically accurate
- Claims should cite sources or best practices
- Outdated content should be marked with deprecation notices

### Inclusivity
- Language should be neutral and professional
- No discrimination based on skill level, background, or identity
- Documentation serves diverse learning styles

### Safety
- Avoid recommending unsafe practices without warnings
- Warn about potential risks or limitations
- Link to official resources for critical topics

## Handling Changes

### Minor corrections (typos, clarity)
- Submit via pull request or GitHub issue
- No approval needed for language/formatting improvements

### Content updates (new features, deprecations)
- Submit pull request with explanation of change
- Reference Discord discussions or RFC if available
- Include version/date context in CHANGELOG.md

### Policy changes (new guidelines, restrictions)
- **HUMAN APPROVAL REQUIRED** — Policy modifications must be reviewed and approved by the staff team before merge
- No AI agents may autonomously modify moderation policy or community guidelines
- All policy changes tracked in CHANGELOG.md with effective date

## Questions?

- Ask in Discord: `#documentation` or `#ai-maxing`
- Open a GitHub discussion: https://github.com/Rethunk-AI/discord-docs/discussions
