# Setup & Operations

User and operator setup guides.

## Prerequisites

- **Node.js** 20+
- **Yarn** (or npm)

## Installation

```bash
git clone https://github.com/Rethunk-AI/discord-docs.git
cd discord-docs
yarn install
```

## Running

```bash
yarn dev      # Start development server
yarn build    # Build for production
yarn lint     # Run linter
```
